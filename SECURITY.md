# Kebijakan Keamanan

## Melaporkan Masalah Keamanan

Jika Anda menemukan celah keamanan atau masalah keamanan lainnya dalam proyek ini, mohon kirimkan laporan ke email: m.fajarramadhani00@gmail.com

Mohon untuk **tidak** membuka GitHub issue untuk masalah keamanan.

## Panduan untuk Laporan

Mohon sertakan detail berikut dalam laporan Anda:
- Deskripsi masalah keamanan
- Versi proyek yang terpengaruh
- Langkah-langkah untuk mereproduksi masalah
- Potensi dampak dari masalah tersebut
- Saran untuk perbaikan (jika ada)

## Respons Keamanan

Kami berkomitmen untuk merespons laporan keamanan dalam waktu 72 jam (3 hari kerja) sejak dilaporkan dan akan memberikan update secara berkala tentang status perbaikan.

## Versi yang Didukung

Kami hanya memberikan patch keamanan untuk versi terbaru dari proyek ini.
